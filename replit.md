# FreeCodeCamp Information Security Challenges

## Overview
This is a FreeCodeCamp boilerplate for completing Information Security challenges. The application uses Express.js with Helmet for security headers.

## Project Structure
- `myApp.js` - Main application entry point
- `server.js` - API routes and middleware (DO NOT EDIT - used for FCC verification)
- `views/` - HTML templates
- `public/` - Static assets (CSS, etc.)

## Tech Stack
- Node.js
- Express.js
- Helmet (security middleware)

## Running the App
The application runs on port 5000 and listens on 0.0.0.0 for Replit compatibility.

```bash
npm start
```
